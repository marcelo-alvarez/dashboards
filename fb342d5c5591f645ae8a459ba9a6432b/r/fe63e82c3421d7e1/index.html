What this is

The full run (1642 realizations × 12 redshifts = 19,704 boxes) drains on cosmosim in redshift-outer

order, so a complete per-z ensemble lands first. This report posts each redshift's final Hα

luminosity function with the realization-to-realization 1σ / 2σ / 3σ scatter as soon as that z is

done — not a preview, the real thing. The full 24-panel characterization (both lines × 12 z) is **report

#6**, which follows once every redshift is complete.

z = 1.4 — complete (1,642 realizations)

The fiducial redshift, the first full ensemble down. Mean abundance-matched to Nagaraj+2023; the

nested bands are the real realization scatter (cosmic variance + shot noise across the 500 Mpc boxes).

Covariance

The realization-to-realization covariance of φ(L), shown as the normalized correlation matrix on a

symlog color scale (linear within ±0.01, logarithmic beyond) so the weak off-diagonal structure —

invisible on a linear ±1 scale — is resolved.
