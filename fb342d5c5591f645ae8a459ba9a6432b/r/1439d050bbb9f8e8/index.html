The headline view: the Hα luminosity function at z=1.4 — the fiducial redshift, which drained

first and so has the most boxes — with the realization-to-realization 1σ / 2σ / 3σ scatter over all

What this is

The live 100-realization run is still draining. This preview reads only what is already on disk

(nothing cancelled, staged, or submitted — read-only, non-disruptive), runs the exact rehearsal

pipeline on it, and renders the 24-panel characterization so the deliverable's appearance is visible

early.

  • Per redshift, every realization that has that redshift complete is included (a realization need

not have all 12 z). N(z) is annotated on each panel. Current N per redshift: z=0.2:9, z=0.25:9, z=0.3:9, z=0.4:9, z=0.5:9, z=0.8:9, z=1.1:9, z=1.4:9, z=1.7:9, z=2:9, z=2.5:8, z=3:8.

  • Calibration is the real one, built on this subset treated as the full ensemble: ensemble

rank-order abundance matching per redshift → single L_cal(L_in, z) per line → applied to each

realization's LF.

  • Hα → Nagaraj+2023 (observed Hα+[NII]) at the in-band redshifts (0.5 ≤ z ≤ 2.0); out-of-band Hα

is the identity map, labeled uncalibrated.

  • [OII] → Comparat+2016 (evolving Schechter, observed [OII], 0.18 ≤ z ≤ 1.3) and Khostovan+2015

(HiZELS, interpolated) for z > 1.3. Both lines show the prior uncalibrated mock LF (dotted) for

reference.

  • 24 panels per the deliverable: ensemble-mean LF vs target, individual realizations, 68%/95% scatter.

How to read it / caveats

  • N is small (~8–9 per panel) — the mean and individual realizations are meaningful, but the 68%/95%

bands are indicative only; they will tighten and fill in with the full 100.

  • The Hα in-band means sit on Nagaraj by construction (that is what abundance matching enforces); the

informative content is the realization-to-realization scatter and the bright-end behaviour.

  • The [OII] target stitches Comparat+2016 (z≤1.3) to Khostovan+2015 (z>1.3); the small step at the

z≈1.3→1.4 boundary is the survey-to-survey offset, expected, and will be documented in the final.

  • Out-of-band Hα has no observed target and stays uncalibrated.
